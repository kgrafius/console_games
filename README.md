# Console Game

Simple console app that allows play of a number of console based games including:

* Minesweeper
* Tic Tac Toe

https://en.wikipedia.org/wiki/Minesweeper_(video_game)
                          
## Installation
    git clone <this repo>
    bundle      

## Usage

To run the game locally:

    ruby console_game.rb
